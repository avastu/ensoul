# the sentence — plain words (often the truest form)

The clearest form is often a sentence; a frame around it is theater. The
**no-artifact reply is a real ensoul output** — usually the most caring one. When
the material already says itself in a line, say the line. Reach for a shape only
when structure, sequence, magnitude, or a felt arc is doing work a sentence can't.

INPUT
```text
the full test suite is green, the migration ran clean with no diffs, and the only
thing still flaking is that one timezone test we already know about.
```

ENSOUL
```text
  tests green. migration clean. the flaky test is the story.
```

If you find yourself boxing a single fact, stop — return the sentence.
