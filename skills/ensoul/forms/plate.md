# plate — an ordinary thing with a hidden center (wild)

**Nerve:** a day, a list, a set of obligations is not flat — it has a *center of
gravity*, and the rest orbits it. **Material:** ASCII plate grammar — a contained
field, a named center, satellites arranged by felt relation (distance = pull). This
is the `wild` move when the input is small but wants presence: don't make a timeline
with costume jewelry — compose the actual forces so the reader feels what pulls what.
One clear center; break symmetry only where the tension is real.

**Plate vs its wild siblings:** use **plate** when *concrete things* (tasks, people,
events) arrange around a center — drawn, named, placed by hand. Use **calligram**
when a single *act* intensifies inward (attention, listening). Use **terrain/field**
for a numeric series or force. "My week orbits Friday" is a plate, not a calligram.

INPUT
```text
today: gym in the morning, lunch with sam around noon, groceries on the way home,
call mom at some point, and the big quarterly report that everything else is
dancing around.
```

ENSOUL
```text
  +-- PLATE / THE DAY HAS A CENTER ----------------------+
  |                                                      |
  |        gym                         call mom          |
  |          \                           /               |
  |           \                         /                |
  |            \      THE REPORT       /                 |
  |             \    +------------+   /                  |
  |              ----| focus now  |----                  |
  |                  +------------+                      |
  |                /                \                    |
  |        lunch with sam        groceries               |
  |             noon             way home                |
  |                                                      |
  |   one center; the rest becomes orbit                 |
  +------------------------------------------------------+
```
