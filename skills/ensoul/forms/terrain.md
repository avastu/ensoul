# terrain — a series as a landscape (wild)

**Nerve:** a felt arc — a launch week, an effort that built and broke, a load that
swelled then settled. The point is the *shape of the rise*, not the digits. **The
gold wild form:** `terrain()` renders a real series as a filled, shaded profile —
a striking image that is ALSO a truthful area chart (the surface IS the data, deeper
fill reads darker). Only on the `wild` dial; one dominant image, facts in a caption
beneath. Generated, so it's dense and exact where hand-typed art goes timid or drifts.
Tuned for dark terminals (more ink = heavier) — say so when you ship one.

INPUT
```text
launch week. monday froze the code. tuesday a staging fire. wednesday the
all-nighter. thursday afternoon we shipped. friday the metrics came in green.
exhausting, but it built and built to that one moment, then breathed out.
```
