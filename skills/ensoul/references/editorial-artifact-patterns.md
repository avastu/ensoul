# Editorial ARTifact Patterns

Use this reference when the goal is a memorable visual object, not merely a correct chart. It blends Tufte-style integrity with Works in Progress / Stripe Press-inspired editorial art direction, mycelial network thinking, and grounded psychomagic.

## Core Doctrine

An ARTifact is a designed object that lets the viewer understand by seeing. It may include words, but words should sit inside the visual field as labels, captions, legends, marginalia, specimens, or evidence traces. The composition should carry the cognition.

Dense means:
- many relationships visible at once;
- meaningful contrast between regions, layers, scales, or forces;
- a picture that can be reread;
- a memory palace for an idea.

Dense does not mean:
- walls of tiny text;
- decorative complexity;
- an ornate skin over a weak argument;
- a beautiful fog that hides missing evidence.

## WIP / Stripe Press Translation

Observed principles from Works in Progress and the "Make it Beautiful" design talk:

- **Ornament can be information.** A wagon, etching, map, texture, specimen, or diagram changes the reader's cognitive mode when it is meaningfully tied to the subject.
- **White space is not automatically clarity.** Empty space can be skipped; meaningful visual matter can slow the eye and situate the idea.
- **Each article gets a world.** Different ideas should not be flattened into the same generic visual language.
- **Archives are living material.** Historical engravings, classified ads, old type, diagrams, ledgers, scientific images, and book ornaments can feel fresh when recontextualized.
- **Beauty makes difficulty hospitable.** Risk, ambiguity, and puzzle can invite rather than alienate when the surface is rewarding and the underlying symbol system is fair.
- **Design should not be trapped by user stereotypes.** Build the richest practical experience, not the blandest expected one.

Do not copy WIP's exact visual identity. Borrow the stance: beautiful public cognition, topic-specific worlds, progress-minded seriousness, and ornament that earns its place.

## Mycelial Map Grammar

Use this for complex living systems: strategy, agent networks, communities, memory, creative ecosystems, product surfaces, revenue paths, identity work, and hidden dependencies.

```text
                         fruiting bodies
                 visible outputs / offers / artifacts
                              ▲
                              │
nutrients ───────▶ hyphae ────┼────▶ symbiosis
money trust       traces      │      alliances / mutuality
attention time    paths       │
evidence energy   dependencies│
                              ▼
                         decomposition
                  stale beliefs / debt / rot / release
```

Mark vocabulary:

| Mark | Meaning | Use |
| --- | --- | --- |
| `o` | node | person, idea, artifact, decision, proof |
| `--` | visible link | direct dependency or sequence |
| `..` | latent link | possible path, weak signal, intuition |
| `=>` | nutrient flow | resource moves through the system |
| `xx` | rot | blocked flow, stale premise, shame loop, waste |
| `*` | fruiting body | visible output or high-leverage artifact |
| `()` | protected substrate | hidden context, roots, memory, trust |

## Psychomagic Mirror Pattern

Use for internal conflict, values, identity, strategic ambiguity, relationship questions, or "what am I not seeing?"

Keep it grounded:
1. Name the real forces.
2. Give each force a symbol, location, or material.
3. Show the pattern of attraction, avoidance, exchange, blockage, or transformation.
4. Add one small action or ritualized move that changes the field.
5. Keep an evidence drawer underneath: what in the prompt or context justifies this image?

Template:

```text
                 visible wish
                     *
                    / \
         old vow ..o   o.. new demand
                  /     \
              xx fear   $ material pressure
                  \     /
                 (root memory)
                      │
                 next honest move
```

## Editorial Plate Pattern

Use this for public-facing explainers, strategy notes, research summaries, and product narratives.

```text
TITLE AS CLAIM
short subtitle that names the mechanism

[dominant image / map / symbolic field / annotated object]
  callout 1: evidence
  callout 2: mechanism
  callout 3: consequence

margin: historical echo / glossary / tiny timeline / source trace
footer: implication -> next action
```

Design rules:
- One dominant visual world per artifact.
- Use labels as part of the image, not captions banished below it.
- Let historical or ornamental material introduce the subject's atmosphere.
- Put evidence close to the claim it supports.
- Prefer one strange, exact image over five generic icons.

## Artifact Families

| Need | Form | Strength |
| --- | --- | --- |
| Explain a mechanism | annotated machine / organism / process plate | makes causality visible |
| Compare alternatives | small-multiple worlds | preserves difference without overexplaining |
| Map a living system | mycelial field | reveals hidden dependencies and flows |
| Surface a tension | psychomagic mirror | turns vague conflict into inspectable shape |
| Public research | editorial plate | makes an underrated idea vivid |
| Product/strategy | force-field atlas | shows tradeoffs, constraints, and moves |
| Agent behavior | organ table + network map | links inputs, authority, outputs, failure smells |

## Integrity Tests

- If the ornament disappeared, would understanding get worse? If not, remove it.
- Can the viewer retell the structure without reading a paragraph?
- Does beauty increase patience with real complexity?
- Are symbols stable across the artifact?
- Is ambiguity productive, or is it hiding a weak claim?
- Can the human recognize the lived pattern from the artifact itself?
